## 🧠 Conceitos Praticados
- [x] **Encapsulamento**: Uso de `private` e métodos `get/set`.
- [x] **Sobrecarga**: Múltiplos construtores com `this()`.
- [x] **Membros Estáticos**: Uso de `static` para contadores globais.
- [ ] **Composição**: Objetos dentro de objetos com **cópia defensiva**.
- [x] **Vetores de Objetos**: Manipulação de arrays, buscas e médias.
- [x] **Ordenação**: `Sort` alfabético e numérico.
